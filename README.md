# ✦ VASTRA ✦

## ✨ Features

| Feature | Description |
|---|---|
| 🎬 **Cinematic Splash Screen** | Animated brand intro with staggered letter reveal |
| 🖼️ **Parallax Hero** | Full-width hero with scroll-driven depth effects |
| 🛍️ **Product Catalog** | Category-based browsing with elegant grid layouts |
| 📄 **Product Details** | Immersive PDP with size guide, image gallery & add-to-cart |
| 🛒 **Cart Drawer** | Slide-out cart with real-time state via Zustand |
| 📖 **Editorial Lookbook** | Fashion-forward lookbook section with hover reveals |
| 💬 **Testimonials** | Customer reviews with scroll-triggered animations |
| 📧 **Newsletter Capture** | Email subscription section with glassmorphism styling |
| 📊 **Animated Counters** | Stats that count up when scrolled into view |
| 📱 **Fully Responsive** | Pixel-perfect on mobile, tablet & desktop |
| 🔄 **Page Transitions** | Smooth Framer Motion route transitions |
| 🎯 **Scroll Progress** | Top-bar scroll indicator for UX polish |

---

## 🛠️ Tech Stack

```
Frontend       →  React 19 + TypeScript
Build Tool     →  Vite 8
Styling        →  Tailwind CSS 4 + Custom CSS Variables
Animations     →  Framer Motion + GSAP
State Mgmt     →  Zustand
Routing        →  React Router DOM v7
Icons          →  Lucide React
Deployment     →  GitHub Pages
```

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/suleshcreates/vastra.git

# Navigate to project
cd vastra

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## 🎨 Design Philosophy

VASTRA follows a **luxury minimalism** approach — combining rich textures, warm earth-tone palettes, elegant serif typography (*Playfair Display*), and purposeful micro-animations to create a premium brand experience that feels curated and intentional.

**Color Palette:**
- 🟤 `--color-ink` — Deep charcoal base
- 🟡 `--color-gold` — Warm gold accents
- 🔴 `--color-burgundy` — Rich burgundy highlights
- ⚪ `--color-sand` — Soft neutral backgrounds
